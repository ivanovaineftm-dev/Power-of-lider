# Excel Processor Web App

Локальное веб-приложение для загрузки, обработки и выгрузки Excel-файлов.

## Возможности

- Три файлоприёмника:
  - `СЛ`
  - `Штат актуальный`
  - `Штат на период`
- Поддержка форматов `.xlsx` и `.xls` при загрузке.
- Очистка каждого слота и повторная загрузка.
- Обработка файла на сервере.
- Выгрузка обработанного файла.

## Логика обработки

При обработке в конец таблицы добавляются 4 столбца:

1. `Должность по состоянию на 05.02.2026`
2. `Истина`
3. `Подразделение по состоянию на 05.02.2026`
4. `Истина`

Значения по умолчанию:

- для колонок должности/подразделения — пустая строка,
- для колонок `Истина` — `True`.

## Запуск локально

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload
```

После запуска откройте:

- http://127.0.0.1:8000

## Структура

- `main.py` — FastAPI backend.
- `static/index.html` — интерфейс.
- `static/app.js` — логика UI.
- `static/styles.css` — стили.
- `storage/uploads` — загруженные файлы (создаётся автоматически).
- `storage/processed` — обработанные файлы (создаётся автоматически).
